# Resumo técnico PC3

O projeto atingiu no PC3 a integração funcional entre:

- sensor PAJ7620;
- ESP32-C3 SuperMini;
- comunicação I2C;
- comunicação Bluetooth BLE;
- iPad;
- interface holográfica.

O sistema reconhece gestos e envia comandos de teclado via BLE para o iPad.
A reprodução holográfica encontra-se em refinamento.
