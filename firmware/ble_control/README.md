# Controle Bluetooth BLE

O ESP32-C3 é configurado como um teclado Bluetooth usando `BleKeyboard.h`.

Nome anunciado no Bluetooth:

```txt
Controle Gestos ESP32
```

Mapeamento usado:

| Gesto | Tecla enviada |
|---|---|
| Direita | 1 |
| Esquerda | 2 |
| Cima | 3 |
| Baixo | 4 |
| Aproximar | 5 |
| Afastar | 6 |
| Horário | 7 |
| Anti-horário | 8 |
```
